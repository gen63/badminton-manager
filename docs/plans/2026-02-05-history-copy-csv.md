# 履歴コピー機能のCSV形式改善

## 概要

試合履歴のコピー機能を、テキスト形式からCSV形式に変更する。

## 変更点

### 1. CSV形式への変更

**Before:**
```
2026/02/05 ぴいす
1 田中 佐藤 21-15 鈴木 高橋
2 田中 鈴木 19-21 佐藤 高橋
```

**After:**
```
日付,場所,A選手1,A選手2,B選手1,B選手2,スコアA,スコアB,試合時間
2026-02-05,ぴいす,田中,佐藤,鈴木,高橋,21,15,12
2026-02-05,ぴいす,田中,鈴木,佐藤,高橋,19,21,10
```

### 2. 日付の取得元変更

- `new Date()` → `session.config.practiceDate` を使用
- セッションがない場合のみ当日日付をフォールバック

### 3. コピーボタンの無効化

- 試合履歴が空のときはコピーボタンを `disabled` にする
- `icon-btn` に disabled スタイルを追加（opacity: 0.4, cursor: not-allowed）

### 4. トースト通知

- 成功時: トースト表示なし（クリップボードにコピーされたことは暗黙的）
- 失敗時: エラートースト「コピーに失敗しました」

## 対象ファイル

- `src/pages/HistoryPage.tsx` - コピーロジック
- `src/index.css` - icon-btn disabled スタイル
